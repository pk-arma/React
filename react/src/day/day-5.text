Uncontrolled Forms

Uncontrolled forms are inputs whose values are managed by the DOM, not React.

Key Points

Value stored internally by the input element.

Accessed using ref.

React does not control the value.

Useful for simple or performance-sensitive forms.

Controlled Forms

Controlled forms are inputs whose values are managed by React state.

Key Points

Value is stored in React state.

Updated using onChange handlers.

React always knows the current input value.

Easy to add validation, formatting, or conditional logic.
